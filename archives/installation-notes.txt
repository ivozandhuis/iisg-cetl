On Debian Trixie you can install the packages in a virtual environment.
```sudo apt install python3-venv python3-pip
python3 -m venv myenv
source myenv/bin/activate
pip install <package>```

Then run ```python3 -m pip install <package>```

- sqlite3
- os
- datetime*
- pathlib*
- csv
- json
- requests*
- Sickle (OAI-PMH library)*
